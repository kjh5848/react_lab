    ### 02. props/ex09

       ### 03. useEffect/ex12
       - 가령 api 요청이 useState가 일어날때마다 리렌더링이 일어나서 계속 api요청이 된다면 어떻게 될까? 렌더링을 한번만 되게 하는 useEffect를 통해 반복적으로 실행되는 렌더링을 컨트롤하고 원하는 값이 변경될때만 렌더링을 하는 방법도 배워보자.
